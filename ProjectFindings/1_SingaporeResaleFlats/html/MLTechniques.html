Univariate Linear Regression

In this section, Floor Area is used to predict Resale Price

One of the simplest machine learning model is the Linear Regression model. Linear Regression is used to predict the value of a variable based on the value of another variable. In order for the linear model to make accurate prediction, the relationship between the 2 variables must be linear. To quantify a linear relationship, Pearson Correlation Coefficient is commonly used. When all data points lie on the fitted linear line, then correlation will be +/- 1. (i.e perfectly positive/negative correlated). Moreover, confidence in inferences can be determine using p-value. Generally, lower p-value suggest greater confidence. Lower p-value can be achieved with more data points.

The best fit linear regression line is obtained via an iteration process. An arbitrary line is first drawn and the Residual Sum of Squared is calculated. Residual is nothing but the distance between the data point and the line drawn. The difference is squared to eliminate negative values and the results of all these differences are summed together, thereby arriving at the name - Residual Sum of Square.

The Pearson's correlation between Floor Area and Resale Price is 0.7685, a relatively positive correlation. The Pearson's correlation is calculated using Covariance(x,y) / (var(x) * var(y)). The Covariance is calculated using sum((x - xbar)(y - ybar)) / (n-1). The purpose of Pearson's correlation is to simply quantify linear relationship between the 2 variables. Therefore, Pearson's correlation does not touch on how well linear line can predict results. This can be performed through Explained Variance R2.

The explained variance for this linear regression line is 0.59. Explained Variance is given by (var(mean) - var(line)) / var(mean) where var(mean) = sum((x - xbar)2) and var(line) = sum(x - xline)2. Explained variance is a metric to discuss how well line is fitted to our data. Hence, an R2 of 59% suggest that 59% of the variation of Resale Price is explained by floor area. This is quite expected since from the image, most datapoints are not within the line. Hence, linear regression may not be suitable here for prediction.

Multivariate Linear Regression

In this section, Floor Area and Remaining Lease will be used to predict Resale Price

.

Correlation between Remaining Lease and Resale price is 0.142. This suggest that the linear correlation is not very strong. This relatively weak correlation can be observed from the scatterplot, whereby change in any specific resale price is spread across remaining lease. Plotting a 3D Scatterplot with Linear Regression line is not as straightforward, especially with 85000 data points.

As expected, the prediction did not improve a lot. The Explained Variance is 0.592, a meagre increase of 0.02. The Mean Squared Error is very high at 6 billion. This is due to the fact that there are many data points (85000+) and resale prices are generally large in magnitude. The RMSE is at 80,151, suggesting that data points are generally $80,000 away from linear regression line.

Conclusion

Linear Regression will not be a very useful model here to predict Resale Prices.

Regression Tree

Regression Tree is explored to determine if it can be a good model for predicting Resale Price. Regression Trees may be a better model to fit features with non-linear relationships. The features used for predicting Resale Price are:

  • town
  • flat_type
  • storey_range
  • floor_area
  • flat_model
  • remaining_lease
In order to train a Regression Tree model using Scikit-Learn, some of the variables have to be label encoded since they are in string format. Head over to Data Preparation to find out more.

The first step to performing Regression Tree is to find the optimum Depth. The optimal depth determined in this scenario was 15, whereby the accuracy plateaus at about 85%. The performance of the Regression Tree was also not very optimal. The MSE was calculated to be 6,109,905,343. That is only a mere improvement of 300,000,000 from the linear regression model.
