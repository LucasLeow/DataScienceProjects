Data Preparation

Data Source

Data was stored in 5 separate csv files.
  1. Resale Flat Data from 1990 - 1999 (287,196 entries)
  2. Resale Flat Data from 2000 - 2012 (369,651 entries)
  3. Resale Flat Data from 2012 - 2014 (52,203 entries)
  4. Resale Flat Data from 2015 - 2016 (37,153 entries)
  5. Resale Flat Data from 2017 onwards (130,404 entries)

Handling Missing Values

Owing to large number of entries, only resale flat data from 2012 onwards was used to speed up data processing. This reduces the number of entries from 876,607 to 219,760. Moreover, resale prices from earlier years are much lower compared to current market prices. The usage of older data will be covered in future sections, for the purpose of pattern / trend discovery.
